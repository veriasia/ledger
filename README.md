# ledger
Veri Asia | Performance Ledger

This repository contains a verifiable history of probability outputs for the J-League, A-League, K-League, Saudi Pro League and China Super League.

Metrics Tracked:
Brier Skill Score (BSS): Measuring accuracy against market benchmarks.
Closing Line Value (CLV): Tracking our 'Tick Move' against sharp bookmaker lines.

Settled Results: Full transparency on model performance.
